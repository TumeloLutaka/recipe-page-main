### What I learned

I learned to use media queries to achieve different visuals based on screen width. I also learned the importance of structuring content when it comes to how the final result needs to be manipulated with media queries. Specifically when it came to the image needing to take up the entire width of the screen in the mobile design.

I plan on focusing on layout and responsive layouts in future.
